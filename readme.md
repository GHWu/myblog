#个人博客----基于nodejs
###支持：express框架+bootstrap+mongodb数据库+各种npm包

###目前实现功能：
1. 个人博客首页按时间排序显示六条博客的缩略，已加入翻页。  
2. 博客的具体内容页面，显示这条博客的全部内容，显示评论。 
3. 发布博客的页面，纯文字。
4. 用户注册登录。
5. 用户发表评论。
6. 侧边栏二维码 返回顶部功能。
7. 后天管理博客、评论、用户。
8. 用户注册邮箱验证码发送验证。
9. 用户修改密码。
10.发布文章可编辑样式。
11.文章的修改。
12.加入点线蜂巢
